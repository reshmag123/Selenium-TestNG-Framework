# Selenium TestNG Automation Framework

This is a **Selenium-based automation testing framework** built using **Java**, **TestNG**, **Maven**, and **Apache POI** for Excel handling.  
It is designed to demonstrate automated UI testing for web applications in a clean, maintainable, and scalable way.
